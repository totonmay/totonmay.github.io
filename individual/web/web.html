One of My First Javascript Projects...

One of the first things I learned how to do with Javascript was creating a web page that prompts the user. When "prompt me" is clicked, a pop up will ask the user in input a username and password. The page will then display the username and password that was entered. If nothing was entered, the page will return "null."

I later learned how to do more complicated things, such as making a score keeper. This score keeper will increase the counter for the player that is selected. The counter keeps score up to 5 points, and when it is reached, the winner will be highlighted green. Hitting reset will clear the scores.

Another type of counter I created was one which changes value based on the time accumulated. This counter increases by 3 values every 3 seconds that passes.

My Dashboard

I created a main dashboard that holds all of my lab work for a past class. We were allowed to go all out, so naturally, I had to stylize it according to my preference.

The dashboard is accessed through a log in page where hitting submit will take you to the main site. Checking "remember me" will save your info for the next time you log in.

The main page contains a navbar which links to their respective websites, and a drop down menu allows one to access to all other labs. The home page is a carousel which automatically slides over to the next image every 7 seconds.

My Portfolio

This portfolio was one of my personal projects where I showcased all of my design work. This was the first "complicated" web page that I had created compared to everything I had done in the past, so it was a pretty big project for me.
The home page of the portfolio has a navbar leading to different parts of the website, and the main text has a typing animation where the lavender text goes through a list of 4 different words.

Selecting "about" in the navbar scrolls you down to the section on the page where I introduce myself.

"Skills" takes you to a page which showcases my knowledge on different languages and programs.

"Work" leads you to thumbnails which link to different web pages showcasing my projects and designs.
